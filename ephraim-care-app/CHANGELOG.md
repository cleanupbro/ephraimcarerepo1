# Changelog

All notable changes to the Ephraim Care website will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [2.3.1] - 2026-01-16

### Changed
- Capitalized slogan: "Embracing Differences" (proper title case)
- Version sync: package.json, footer, and CHANGELOG now aligned
- Dynamic version import in footer component

### Added
- Git tags for version history (v2.1.0, v2.2.2, v2.3.0, v2.3.1)
- GitHub Releases for each version
- Root README.md for GitHub repository

## [2.1.0] - 2026-01-14

### Added
- `/faq` page with NDIS logos and dedicated FAQ section
- Unique hero images for each page (no duplicates)
- Ken Burns animation effect on hero images
- NDIS badge with pulse animation on homepage
- HowItWorks section moved to Referrals page

### Changed
- Contact page hero: `personal-care-hero.jpg` (support worker interaction)
- Referrals page hero: `Support-Community-Auslife-Care.jpg` (community activities)
- FAQ page hero: `NDIS-Disability-Support-Services-scaled.jpg` (group support)
- All backgrounds now use mint gradient (`#E8F5F3` to `#F5FAFA`)
- Replaced all emojis with Lucide icons throughout site
- Hero section simplified - removed trust stats row
- Green color scheme consistent across all pages

### Removed
- Accessibility toolbar (FloatingButtons)
- Trust badges from hero section (moved NDIS badge to bottom)
- Emojis from FAQ questions, How It Works steps, NDIS calculator

### Fixed
- 404 errors for missing hero images
- Cream/white backgrounds replaced with mint gradient
- Blue (#0088CC) colors replaced with green (secondary)
