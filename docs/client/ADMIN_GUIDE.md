# Admin Dashboard User Guide
> How to use the Ephraim Care admin dashboard
> Version: 1.0

---

## Accessing the Dashboard

1. Go to: https://www.ephraimcare.com.au/admin
2. Log in with your admin credentials
3. You'll see the main dashboard

---

## Dashboard Overview

The dashboard shows:
- Total referrals count
- Total contacts count
- Recent activity

---

## Managing Referrals

### View Referrals
1. Click "Referrals" in the sidebar
2. See all referrals in a list
3. Click on a referral to view details

### Update Referral Status
1. Open a referral
2. Change status: New → Contacted → In Progress → Complete
3. Click Save

---

## Managing Contacts

### View Contacts
1. Click "Contacts" in the sidebar
2. See all contact form submissions
3. Click to view full message

### Respond to Contacts
1. View the contact details
2. Note the email/phone
3. Respond via email or phone
4. Mark as "Responded"

---

## Managing Participants

### Add Participant
1. Click "Participants"
2. Click "Add New"
3. Fill in details:
   - Name
   - NDIS number
   - Contact information
   - Support needs
4. Click Save

### View Participant
1. Click on participant name
2. See full profile
3. View appointments
4. View progress notes

---

## Appointments

### Schedule Appointment
1. Click "Appointments"
2. Click "New Appointment"
3. Select participant
4. Choose date and time
5. Add notes
6. Save

### View Calendar
- Calendar view shows all appointments
- Color coded by status

---

## Incident Reports

### Report an Incident
1. Click "Incidents"
2. Click "New Report"
3. Fill in:
   - Participant involved
   - Date and time
   - Description
   - Severity level
   - Actions taken
4. Submit

### NDIS Compliance
- All incidents are logged for NDIS compliance
- High severity incidents trigger alerts

---

## Shift Handovers

### Create Handover Note
1. Click "Handovers"
2. Click "New Handover"
3. Fill in:
   - Outgoing staff member
   - Incoming staff member
   - Participants discussed
   - Important notes
4. Submit

---

## Staff Management

### Add Staff Member
1. Click "Staff"
2. Click "Add Staff"
3. Fill in details
4. Save
